# Прогнозирование оттока клиентов

Проект сравнивает логистическую регрессию, случайный лес и градиентный бустинг на Iranian Churn Dataset.

## Данные

- исходно 3150 наблюдений;
- удалено 300 полных дубликатов;
- после очистки осталось 2850 наблюдений;
- используются 13 исходных признаков;
- `FN` и `FP` исключены.

## Методика

- стратифицированное разделение 80/20;
- 5-кратная кросс-валидация;
- A/B-тест балансировки;
- подбор порога по OOF-прогнозам;
- независимая тестовая выборка;
- анализ важности признаков.

## Результаты

| Модель | Балансировка | Порог | CV F1 | Test F1 | Precision | Recall | ROC-AUC |
|---|---|---:|---:|---:|---:|---:|---:|
| Случайный лес | Без балансировки | 0.405 | 0.863 | 0.891 | 0.907 | 0.876 | 0.985 |
| Градиентный бустинг | Без балансировки | 0.375 | 0.861 | 0.876 | 0.844 | 0.910 | 0.987 |
| Логистическая регрессия | Без балансировки | 0.250 | 0.681 | 0.657 | 0.559 | 0.798 | 0.928 |

**Лучшая модель: Случайный лес.** Test F1 = **0.891**.
